# java-playground
a repo created for testing and learning different features of java and jvm libraries via exploratory unit tests

currently tested stuff:
- Core (Java 8, 9)
- Concurrency
- Patterns (Design Patterns implementations)
- Libs
  - Guava
  - RxJava
  - Reactor
  - Vavr

To run all tests, type:

```
./gradlew test
```
